## Intro

PSAppDeployToolkit is a versatile solution that streamlines and standardizes the process of software deployment, making it easier than ever to manage your IT environment. It has a comprehensive set of features, such as dynamic logging, user interaction capabilities, and customizable functionality.

## Github Repository

I developed a series of Powershell Application Deployment Toolkit scripts to install applications via Microsoft SCCM for computer imaging.

Github Link ➡️ [PSAppDeployToolkit](https://github.com/caseyjdavis/psappdeploytoolkit)

These packages were used to deploy software on 40,000 computers used throughout the Poudre School District located in Fort Collins, Colorado.
